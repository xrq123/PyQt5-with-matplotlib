Use pyside2 to interact with matplotlib and implement a software interface for drawing line, scatter, and pie charts.

The environment of the program is pyside2.

form2.ui is designed by QtCreator.

chart.py is responsible for completing the drawing.

ui_ Form2. py is responsible for completing the design of the UI module.

main.py is the main program and starts running at main.py.

The demo is running on pycharm.
